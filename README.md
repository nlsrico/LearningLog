## Entry template

- Title:  
- Provider:  
- Type: Course | Specialization | Certificate | Track  
- Date: YYYY-MM  
- Hours/Rigor: e.g., 40 hours, graded capstone, proctored exam  
- Skills: 3–6 keywords  
- Evidence: repo/demo/blog  
- Certificate: verification URL (include ID if available)  
- Outcome: one measurable result or deliverable  
- Notes: quick takeaways, pitfalls, or next steps
